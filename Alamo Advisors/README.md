# Alamo Advisors Website

Modern, SEC-compliant website for Alamo Advisors LP - a premier investment advisory firm in San Antonio, Texas.

## Features

- Clean, institutional design with Instrument Serif and Geist typography
- Fully responsive (mobile, tablet, desktop)
- SEC-compliant content (no testimonials, proper disclosures)
- Contact form with serverless backend
- Team showcase page
- Client portal integration

## Development

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Production server
npm start
```

Visit `http://localhost:3000` to view the site.

## Deployment

This site is configured for Vercel deployment with serverless functions.

```bash
vercel deploy
```

## Color Palette

- Primary: #1e5790 (Deep Blue)
- Secondary: #aba489 (Warm Taupe)
- Background: #ffffff (White)

## Typography

- Body: Instrument Serif
- Headings: Geist
